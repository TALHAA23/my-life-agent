## UI Color Palette

The application uses the following color palette:

- **Lightest**: `#F1F3E0` (Backgrounds, Light accents)
- **Light**: `#D2DCB6` (Secondary elements)
- **Medium**: `#A1BC98` (Borders, Secondary text)
- **Darkest**: `#778873` (Primary text, Buttons)
